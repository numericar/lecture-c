Variable 
    ในภาษา C ตัวแปรจะต้องถูกประกาศขึ้นมาก่อนที่ตะถูกเรียกใชงาน
    โดยมีชนิดข้อมูลดังนี้
        int 
            ใชัสำหรััจจัดเก็บข้อมูลที่เป็นเลขจำนวนเต็ม
            โดยขนาดของ bit จะขึ้นอยู่กับคอมพิวเตอร์ที่เราใช้ โดยอาจมี 16 bit และ 32 bit
        float
            ใช้สำหรับจัดเก็บเลขที่มีทศนิยม
            โดยขนาดของ bit จะเหมือนกับ int
        long
            ใชัสำหรััจจัดเก็บข้อมูลที่เป็นเลขจำนวนเต็มที่มีขนาดใหญ่
        short
            ใชัสำหรััจจัดเก็บข้อมูลที่เป็นเลขจำนวนเต็มที่มีขนาดเล็ก
        double 
            ใช้สำหรับจัดเก็บเลขที่มีทศนิยมที่มีขนาดใหญ่
        char 
            ใช้เก็บตัวอักษร

    ** ทำไมขนาดของ bit จึงขึ้นอยู่กับคอมพิวเตอร์ ?
            เนื่องจากภาษา c ถูกออกแบบมาให้มีความยืดหยุ่นในสภาปัตยกรรมของคอมพิวเตอร์และคอมไพลเลอร์ ซึ่งทำให้สามารถทำงานได้บนฮาร์ดแวร์ที่แตกต่างกันได้
        ซึ่งขนาดของ cpu อ่านมี word size ไม่เท่ากัน เช่น 
            CPU 16 bit      -> ขนาดของ int จะมี 16-bti 
            CPU 32 bit      -> ขนาดของ int จะมี 32-bit 
            CPU 64 bit      -> ขนาดของ int จะมี 32 หรือ 64 bit ขึ้นอยู่กับ compiler
        
        โดยไม่ใช่แค่ int แต่รวมถึงทุก ๆ ชนิดของข้อมูล
