Introduction
    ในการที่เราจะเรียนเขียนโปรแกรมสักภาษา เรามักจะเริ่มต้นด้วย hello world
    ตัวอย่างการ hello world ในภาษา c
        #include <stdio.h>

        int main()
        {
            printf("Hello World\n");
        }

    โดยการเรียกใช้งานชุดคำสั่งที่เราเขียนขึ้นบนระบบปฏิบัติการ unix เราสามารถใช้คำสั่ง cc <file.c> ได้เลย
    เช่น cc hello_world.c
    หากไม่มีอะไรผิดพลาด คอมไฟลเลอร์จะสร้างไฟล์ a.out ขึ้นมา และเราสามารถเรียกใช้งานชุดคำสั่งของที่ถูกคอมไพลแล้ว
    ผ่าน a.out ได้เลย

    โปรแกรมภาษา c ไม่ว่าจะมีขนาดเท่าใดก็ตาม จะประกอบไปด้วย ฟังก์ชัน และ ตัวแปร

    ฟังก์ชัน จะประกอบด้วยชุดคำสั่งสำหรับดำเนินการประมวลผล
    ตัวแปร จะจัดเก็บค่าที่อยู่ระหว่างการตำนวน

    ฟังก์ชัน main เป็นฟังก์ชันพิเศษ ซึ่งจะเป็นฟังก์ชันแรกของโปรแกรมที่จะถูกเรียกใช้งาน 
    และทุก ๆ โปรแกรมจะต้องมีฟังก์ชัน main เสมอ
    โดยฟังก์ชัน main จะทำการเรียกใช้ฟังก์ชันอื่น ๆ ที่เราสร้่างขึ้น หรือฟังก์ชันที่มาจาก library

    จากตัวอย่างของชุดคำสั่งของเรา 
        #include <stdio.h>
    เป็นการบอกคอมไพลเลอร์ว่า ให้รวบรวมชุดคำสั่งจากไลบรารี่ input/output standard'

    โดยปกติแล้วเราจะสามารถส่งค่าให้กับฟังก์ชันได้ และค่าเหล่านั้นถูกเรียกว่า arguments โดยจะอยู่ระหว่าง () ของฟังก์ชัน
        #include <stdio.h>              รวบรวมชุดคำสั่งจาก standard library

        int main()                      สร้างฟังก์ชันขึ้นมาตัวหนึ่งชื่อว่า main โดยไม่รับ agruments ใด ๆ
        {                               ชุดคำสั่งที่ฟังก์ชัน main จะดำเนินการจะอยู่ภายใต้ปีกกาเปิด/ปิด
            printf("Hello World\n");    ฟังก์ชัน main เรียกใช้งานฟังก์ชันที่มาจาก library i/o โดยฟังก์ชันทีชื่อว่า printf
        }

    ฟังก์ชัน จะถูกเรียกใช้งานด้วยชื่อ และตามด้วย arguments ที่ฟังก์ชันนั้นต้องการ จากตัวอย่างคือ printf ที่รับ arguments เป็น "Hello World"
    ซึ่งฟังก์ชันดังกล่าว จะทำหน้าที่นำพิมพ์ผลลัพธ์ออกไปที่ terminal

    ลำดับอักขระในเครื่องหมายคำพูด ("") จะเรียกว่า character string หรือ string constant
