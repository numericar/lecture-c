Feature in c
    - มีขนาดกะทัดรัด
    - มีประสิทธิภาพ
    - ยืดหยุ่น

Efficiency
    เป็นภาษาที่สามารถทำงานได้อย่างรวดเร็ว และมีขนาดกะทัดรัด และโปรแกรมเมอร์สามารถปรับแต่งตัวภาษาได้ดีให้มีประสิทธิภาพสูง
    ให้เหมาะสมกับสถาพแวดล้อมที่กำลังทำงานด้วย และสามารถจัดการ memory ได้เอง

Portability
    สามารถรันบนระบบปฏิบัติการอะไรก็ได้ โดยไม่จำเป็นต้องแก้ไข
    และสามารถทำงานกับคอมพิวเตอร์ทุกสถาปัตยกรรม
    
Power
    compiler ของภาษาอื่น ๆ ส่วนใหญ่เขียนด้วย c เช่น fortram, perl, pascal, lisp, logo, basic
    และภาษา c ยังถูกใช้ในด้านวิศวกรรมอีกมาก เช่น การแก้่ปัญหาทางฟิสิกส์ ปัญหาทางวิศวกรรม การเคลื่อนไหวของเอฟเฟกต์ในภาพยนตร์

Flexibility
    ภาษามีความยิดหยุ่น สามารถสรร้างแอปพลิเคชันได้ทุกประเภท สามารถจัดการ memory ได้ด้วยตัวเอง

Programmer Oriented
    มีความเข้มงวดน้อยกว่าาภาษาอื่น
    มี library ให้ใช้เยอะมาก


* ใน Linux/Unix สามารถรันโปรแกรมภาษา c ได้ทันที เพราะมี compiler อยู่ในตัวอยู่แล้ว
