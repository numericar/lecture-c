Preprocessor
    ตัวประมวลผลร่วงหน้า เป็นคุณสมบัติในภาษา c (ไม่มีในภาษาอื่น)
    โดยจะเป็นชุดคำสั่งพิเศษ ที่จะถูกประมวลผลก่อนชุดคำสั่งหลักของโปรแกรม (คำสั่งที่อยู่ใน main)

    คอมไพล์เลอร์จะดำเนินการประมวลผล preprocessor ก่อนทุกการทำงาน
    โดยชุดคำสั่ง preprocessor มักจะอยู่ด้านบนสุดของโปรแกรม มักจะขึ้นต้นด้วยสัญลักษณ์ (#)
    เช่น 
        - #include <stdio.h>
    โดยมันมีคำสั่งสำหรับการประมวลผลล่วงหน้า ดังนี้
        1. #define  - สำหรับกำหนดค่าคงที่ล่วงหน้า โดยจะเป็นค่าที่ไม่ถูกเปลี่ยนแปรงตลอดการทำงานของโปรแกรม
        2. #include - เป็นการนำชุดคำสั่งจากโปรแกรมอื่น มาใส่ในโปรแกรมของเรา (ทำให้เราสามารถใช้งานชุดคำสั่งที่คนอื่นเขียนไว้อยู่แล้วได้เลย ไม่ต้องเขียนใหม่)
        3. #ifdef, #endif, #else, #ifndef -

Header File
    ใช้จัดเก้บส่วนหัวของโปรแกรม เช่น การประกาศฟังก์ชัน, ตัวแปรส่วนกลาง (Global Variable), ค่าคงที่ (Macros)
    เพื่อให้โปรแกรมอื่น ๆ สามารถเรียกใช้งานฟังก์ชันหรือประกาศเหล่านี้ได้ โดนไม่จำเป็นต้องคัดลองโค้ดไปทั้งหมด
    เช่น 
        เราต้องการแสดงข้อความบร terminal เราไม่จำเป็นต้องเขียนขึ้นมาเอง แต่ใช้งานฟังก์ชัน printf ซึ่งอยู่ stdio.h เข้ามาใช้งาน
    เราสามารถนำเข้า header file มาใส่ในโปรแกรมของเราได้ 2 รูปแบบ
        1. #include <stdio.h>
        2. #include "stdio.h"
